# UI/UX Spec

## Direction

Beautiful, playful, premium, soft but not childish.

The UI should feel like a couple's planning ritual: warm, fast to scan, and slightly celebratory. Avoid generic enterprise dashboards, oversized marketing sections, and dense admin panels.

## Mobile-First Principles

- Design first for a narrow phone viewport.
- Use bottom navigation for the five core screens.
- Keep the primary action within thumb reach.
- Prefer short labels and scannable cards.
- Use generous spacing without wasting the first viewport.

## Visual Language

- Soft off-white base with warm coral, mint, ink, lavender, and butter accents.
- Cards may be used for activities and sessions, but avoid nested cards.
- Radius should stay modest and polished.
- Use tactile chips for budget, tags, and ratings.
- Make dates and week labels clear without relying on week numbers.

## Screen Notes

### Week Board

- First screen after launch.
- Shows This Week / Ongoing first.
- Shows Planning below current week.
- Uses a small timeline feel for ban slots and to-dos.
- Makes outcome actions visible only for current sessions.

### Activity Pool

- Shared list of active ideas.
- Supports budget group, tags, duration, and owner signal.
- Paused ideas stay visible but visually quiet.

### Draw Flow

- Feels playful and deliberate.
- Requires target week selection.
- Shows that accepted results become scheduled plans, not history.
- Has a clear accept action and a clear redraw path requiring agreement.

### History

- Shows only archived outcomes.
- Rating chips are prominent.
- Missed/replaced/redrawn outcomes explain why the item is historical.

### Settings / Pair

- Shows pair name, timezone, members, mock mode, and V0 scope notes.
- Timezone default is `Australia/Melbourne`.

## Copy Tone

- Warm and concise.
- Lightly playful, never childish.
- Avoid explaining app mechanics in large instructional blocks.

## Accessibility

- Maintain readable contrast.
- Use real buttons for actions.
- Keep focus outlines visible.
- Do not rely on color alone for state.
