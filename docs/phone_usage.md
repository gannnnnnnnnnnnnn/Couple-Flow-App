# 手机使用说明

Couple Flow 部署到 Vercel 后，两个人用同一个公开网址就可以一起使用。

## 第一次配对

1. 两个人分别用手机打开同一个部署网址。
2. A 进入 `设置`，点击创建配对码。
3. A 把配对码发给 B。
4. B 在自己的手机上打开同一个网址，进入 `设置`。
5. B 输入配对码和自己的昵称，然后加入。
6. 加入后，两台手机会读取同一个双人空间的数据。

## 日常使用

- 两台手机不需要连同一个 Wi-Fi。
- 电脑不需要一直开着。
- 两个人不需要同时在线。
- 同步需要网络；如果手机离线，等恢复网络后再刷新或继续操作。
- 两个人都可以添加活动、抽签、收下计划、补记录。
- 手机浏览器可以把网站添加到主屏幕，之后就像打开一个小 App 一样使用。

## 小提醒

- 配对码是 V0 的轻量同步方式，不是正式账号系统。
- 如果某台手机想退出当前双人空间，可以在 `设置` 里断开本设备。
- 断开本设备只会清掉这台手机上的配对身份，不会删除云端双人数据。
